## Overview

**Yellow Flash** is a modern, fast, and user-friendly bike rental platform. It allows users to browse, rent, and manage bicycles with an intuitive interface and seamless experience on both desktop and mobile devices.

This project demonstrates my skills in **modern frontend development**, including React, TypeScript, and responsive UI design.

---

## Features

- Browse available bikes with detailed information  
- User-friendly booking system  
- Responsive UI for desktop and mobile devices  
- Modern and maintainable code using **React + TypeScript**  
- Styled with **Tailwind CSS** and **shadcn-ui** components  
- Optimized for fast loading and smooth interactions  

---

## Tech Stack

- **React** – Frontend library  
- **TypeScript** – Type-safe JavaScript  
- **Vite** – Fast build tool and dev server  
- **Tailwind CSS** – Utility-first CSS framework  
- **shadcn-ui** – Component library for consistent UI

## Getting Started

Follow these steps to run the project locally:

```bash
# 1. Clone the repository
git clone https://github.com/Prakashramu-23/yellow_flash_bike_rent

# 2. Navigate to the project folder
cd yellow_flash_bike_rent

# 3. Install dependencies
npm install

# 4. Start the development server
npm run dev
